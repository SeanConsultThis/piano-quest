＋ Add Your Own Songs

Three ways to add any song you like. Added songs appear under “My Songs”.

Upload sheet music (MusicXML)

MusicXML is the standard digital sheet-music format. Where to get it:

  • MuseScore (free app, musescore.org): open any score, then File → Export → MusicXML → Uncompressed (*.musicxml).
  • musescore.com has thousands of free arrangements you can download.
  • Scanning apps like PlayScore 2 can photograph paper sheet music and export MusicXML.

The app reads the melody line (first part, top notes).

Upload a MIDI file (.mid)

MIDI files are everywhere — search for “song name midi” online, or export from GarageBand, Synthesia, or any keyboard app. The app picks the busiest track as the melody and keeps the highest note when chords are played.

Type a melody in

Write notes separated by spaces. Each note is a letter + octave (Middle C is C4):

  • C4 — one beat  •  G4:2 — held 2 beats  •  F#4 — sharp, Bb3 — flat
  • R:1 — a rest  •  C4:1(Twin-) — with a lyric  •  | bar lines are ignored
